# ACoLi Dicts
Ontolex-lemon dictionaries provided by the Applied Computational Linguistics (ACoLi) lab at Goethe Universit√§t Frankfurt am Main, Germany, and the associated research group Linked Open Dictionaries (LiODi, 2015-2020, funded by BMBF)

At the moment, we provide OntoLex-lemon and TIAD-TSV dictionaries for 130 language varieties and 373 language pairs, see statistics below.
Additional data has been converted, but is still awaiting copyright clearance.
